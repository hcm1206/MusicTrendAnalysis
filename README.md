# 음악 트랜드 분석 프로젝트

## 개요
음악의 메타 데이터와 트랜드 지수, 각종 사이트의 언급량과 트래픽 등을 분석하여 음악의 인기도 추세를 예측하는 데이터 분석 프로젝트 레포지토리입니다.  

## Notice
레포지토리 로컬에 받고 가상환경 설정하는 과정을 [이 링크](https://github.com/hcm1206/MusicTrendAnalysis/issues/1)에서 설명해놓았으니 필요하시다면 참고해주세요.
