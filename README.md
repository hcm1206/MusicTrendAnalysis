# 음악 트랜드 분석 프로젝트

## 개요
음악 이전 랭킹과 트랜드 지수, 트래픽 등을 시계열 분석하여 음악의 다음 랭킹을 예측하는 데이터 분석 프로젝트 레포지토리입니다.  

## Notice
서버 실행 시 MySQL DB 구축 및 데이터 입력 작업이 필요합니다.
